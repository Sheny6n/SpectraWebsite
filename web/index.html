/ WEB WALLET · IN DEVELOPMENT

Spectra, in the browser.

A browser-based Spectra wallet is on the way — the same open-source core, the same no-telemetry guarantees, running client-side in any modern browser. No accounts, no tracking, no SaaS.

/ 01

Client-side only

Keys are generated, stored, and signed with entirely in your browser. The server never sees them.

/ 02

Reproducible builds

Every release is pinned, hashed, and signed. Verify what you're running before you run it.

/ 03

Self-host ready

Static bundle. Host it yourself, run it offline, air-gap it. No backend required.

Get notified at launch

One email when the beta opens. No newsletter, no follow-ups, no list resale.

We don't run a mailing list yet — this form is a placeholder until the beta signup opens.
