Found something broken?

Spectra is open source and maintained by contributors in their spare time. Clear, reproducible reports are the single most valuable thing you can send us. No account required — submit below or open a GitHub issue directly.

Report a bug

Nothing is sent to any Spectra server. Submitting opens GitHub with your report pre-filled — you review it before anything is posted.

Before you file

Please check the open issues first — your bug may already be tracked. For suspected security vulnerabilities, do not file a public issue: email security@spectrawallet.org with a PGP-encrypted report.
